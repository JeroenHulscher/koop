De beschikbaarheid van informatieobjecten

Het gebruik van een URI

Om de informatie op Overheid.nl zo optimaal mogelijk te kunnen (her)gebruiken, is het van groot belang dat onze informatieobjecten voorzien zijn van een machineleesbare en persistente identificatiecode in de vorm van een Uniform Resource Identifier (URI). Een URI lijkt op een Uniform Resource Locator (URL) – het adres van een website of een document op een website – maar terwijl een URL kan veranderen, is een URI onveranderlijk en scherp gedefinieerd.
